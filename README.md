全景热点编辑器
==

### logs:

新增代码复制按钮，一键复制生成的代码。

### 功能:
    1.键入热点名称，若热点已存在，进入编辑模式，如热点不存在，提示是否创建热点；
    2.集成常用热点属性：大小（scale）、透明度（opacity）、旋转(rotate)、3D属性（rx、ry、rz）、平移（ox、oy）;
    3.热点事件自定义。可指定跳转场景或自定义；
    4.支持热点文字；
    5.所有属性设置均可视化实时显示；
